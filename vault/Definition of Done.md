# Definition of Done

A successful hackathon demo = a Slack user DMing the bot **three distinct queries**:

1. ✅ A general **HR/IT** question → answered from [[Tool 1 - answer_day_one_logistics]].
2. ✅ A request for **system access** → yields an `@X-Bot` command from [[Tool 2 - get_access_instructions]].
3. ✅ A highly specific **technical** question → summarized answer pulled directly from a Confluence page ([[Tool 3 - search_confluence_wiki]]) or Slack thread ([[Tool 4 - search_slack_history]]), **complete with a source link**.

## Demo script (draft)
> "Where do I find my benefits?"
> "How do I get access to the daily cluster?"
> "What is the architecture of our core service?"
