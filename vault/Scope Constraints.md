# Scope Constraints (Hackathon POC)

Explicitly **out of scope** to ensure a working demo:

- ❌ **Automated provisioning** — bot tells the user *how* to get access (e.g. "type this to X-Bot") but does **not** provision access itself.
- ❌ **Data migration** — backing up / migrating data across machines is deferred.
- ❌ **Global rollout** — data sources and prompts are hardcoded/scoped strictly to a **single pilot team**.
